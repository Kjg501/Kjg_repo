# Markdown 사용법
참고 : "https://www.markdownguide.org/", Markdown Guide

## 1. Markdown 문법

### 1.1 제목 (Headings) 
- #으로 시작
- #은 1 ~ 6 가능
- #이 늘어날때마다 제목의 크기가 작아짐
- #한칸띄우고 작성

# #이 한개입니다.
## #이 2개입니다.
### #이 3개입니다.
#### #이 4개입니다.
##### #이 5개입니다.
###### #이 6개입니다. 
#######7개 부터는 지원하지 않습니다.
